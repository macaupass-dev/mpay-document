## 會員卡詳情接口（卡詳情）

### 說明

會員卡詳情信息;所有的卡信息全部集中在卡详情接口中，包括會員資料，若用户修改会员资料，也是先调用卡详情接口，獲取會員資料。

### 需要登錄

是

### 請求參數

#### 公共參數

|     參數     |              值               |
| :----------: | :---------------------------: |
|   api_name   | mpay.member.detail.card.query |
| biz_api_code |           110120005           |

#### 業務參數

| 參數         | 字段名               | 必填 | 字段類型 | 長度 | 備註                 |
| ------------ | -------------------- | ---- | -------- | ---- | -------------------- |
| cust_id      | 客户编号             | 是   | String   |      | 客户编号             |
| mpay_card_no | Mpay卡號             | 是   | String   |      | Mpay卡號             |
| open_app_id  | 會員卡對應的商戶編號 | 是   | String   |      | 會員卡對應的商戶編號 |
|              |                      |      |          |      |                      |

#### 詳情欄目列表說明

| 參數       | 字段名   | 必填 | 字段類型         | 長度 | 備註                                                         |
| ---------- | -------- | ---- | ---------------- | ---- | ------------------------------------------------------------ |
| item_name  | 條目名稱 | 是   | String           |      | 條目名稱：活動、諮訊、會員資料、會員權益                     |
| item_value | 條目連接 | 否   | String/JSONARRAY |      | 類型為 HYPERLINK、HTML 时对应String<br />類型為 MEMBER_INFO、MEMBER_BENEFIT 時為JSONARRAY |
| item_type  | 條目類型 | 是   | String           |      | 條目類型，取以下枚舉值：<br/>HYPERLINK-轉到超級鏈接（H5頁面）；<br/>HTML-展示HTML內容；<br/>MEMBER_INFO-展示會員資料頁（原生頁面）；<br/>MEMBER_BENEFIT-展示會員權益頁（原生頁面）； |
|            |          |      |                  |      |                                                              |
|            |          |      |                  |      |                                                              |

| 參數          | 字段名           | 必填 | 字段類型 | 長度 | 備註           |
| ------------- | ---------------- | ---- | -------- | ---- | -------------- |
| cust_id       | 客戶編號         | 是   | String   |      | 客戶編號       |
| user_name     | 中文名稱         | 否   | String   |      | 中文名稱       |
| foreign_name  | 外文名稱         | 否   | String   |      | 外文名稱       |
| gender        | 性別             | 否   | String   |      | 性別           |
| cert_no       | 證件號碼         | 否   | String   |      | 證件號碼       |
| brithday      | 出生日期         | 否   | String   |      | 出生日期       |
| phone_area    | 區號             | 否   | String   |      | 區號           |
| phone_no      | 電話             | 否   | String   |      | 電話           |
| gender_need   | 性别是否必填     | 是   | int      |      | 0 非必填 1必填 |
| phone_no_need | 电话是否必填     | 是   | int      |      | 0 非必填 1必填 |
| brithday_need | 出生日期是否必填 | 是   | int      |      | 0 非必填 1必填 |
| cert_no_need  | 證件號碼是否必填 | 是   | int      |      | 0 非必填 1必填 |
|               |                  |      |          |      |                |



#### 會員權益列表說明

| 參數    | 字段名   | 必填 | 字段類型 | 長度 | 備註                                                         |
| ------- | -------- | ---- | -------- | ---- | ------------------------------------------------------------ |
| title   | 權益標題 | 是   | String   |      | 權益標題                                                     |
| content | 權益內容 | 是   | String   |      | 權益內容                                                     |
| type    | 權益類型 | 否   | String   |      | remark:使用須知 contact:聯繫方式 validty:有效期 benefit:權益說明 |
