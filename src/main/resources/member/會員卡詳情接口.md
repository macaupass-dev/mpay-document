#### 詳情欄目列表說明

| 參數       | 字段名   | 必填 | 字段類型 | 長度 | 備註                                                         |
| ---------- | -------- | ---- | -------- | ---- | ------------------------------------------------------------ |
| item_name  | 條目名稱 | 是   | String   |      | 條目名稱：活動、諮訊、會員資料、會員權益                     |
