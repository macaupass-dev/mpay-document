## 會員卡列表查詢

### 說明

卡包獲取會員卡列表;從綁卡入口（九宮格）獲取

### 需要登錄

是

### 請求參數

#### 公共參數

|     參數     |          值           |
| :----------: | :-------------------: |
|   api_name   | mpay.member.card.list |
| biz_api_code |       110120002       |



#### 業務參數

| 參數    | 字段名   | 必填 | 字段類型 | 長度 | 備註     |
| ------- | -------- | ---- | -------- | ---- | -------- |
| cust_id | 客户编号 | 是   |          |      | 客户编号 |
|         |          |      |          |      |          |
